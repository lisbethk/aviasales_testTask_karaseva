Проект написан на архитектуре MVVM
Побит на три слоя - core layer, service layer(сервисы) и presentation layer(экраны приложения).
Навигация в приложении написана на UIKit, UI написан на SwiftUI (навигация у SwiftUI страдает)
Приложение запускается из MainFlowCoordinator, зависимости лежат в ApplicationAssembly
UI собирается из моделек, генерируемых фабриками экранов
Проект iOS 14+

Массив рейсов получен с сервера с помощью completion handler'ов', и отсортирован по возрастанию цены
Модели билетом парсятся по протоколу Codable
У самого дешевого рейса бейдж "самый дешевый", для рейса с менее десятью билетами есть надпись об этом
По тапу на рейс отображаются детали рейса, по нажатью на кнопку "купить билет за х" - показывается alert "куплен билет за х"

Во время загрузки билетов крутится loader, в случае получения ошибки отображается текст "что то пошло не так" локализованный текст ошибки, и кнока "повторить" 
Приложение поддерживает темную тему с помощью системных цветов, которые становятся менее яркими в темной теме. (кроме цвета кнопки "купить билет за Х" - это кастомный цвет, работающий через наборы цветов)

Есть два модульных теста, времени не хватило на большее количество, лежат в корневой папке
